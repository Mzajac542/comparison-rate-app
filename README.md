# Odds Comparison App

A web application for comparing betting odds across bookmakers.

## Features
- Browse sports, leagues and matches
- Compare odds from multiple bookmakers
- Highlight highest and lowest odds
- Top 5 best betting opportunities
- Responsive UI

## Tech stack
- Node.js
- Express
- Vanilla JavaScript
- HTML / CSS

## How to run locally
```bash
npm install
node src/server.js
